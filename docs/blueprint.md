# **App Name**: Venue Vendors

## Core Features:

- Vendor Profiles: Allow vendors to create profiles with detailed descriptions and photos of their services, categorized by service type. Include a service category for venues.
- Vendor Search & 'My Favs': Enable clients to search for vendors by state, city, and category, with a 'My Favs' list to save preferred vendors. Implement a 'No vendor found' sharing feature.
- User Authentication: Implement user authentication with separate account types for clients and vendors.

## Style Guidelines:

- Primary color: Purple (#800080) to establish the brand identity.
- Secondary colors: Light Lavender (#E6E6FA) and a soft gray (#D3D3D3) for backgrounds and text.
- Accent: Teal (#008080) for interactive elements and call-to-action buttons.
- Header with purple background and confetti design, including sign-up and log-in buttons.
- Use heart icons for the 'My Favs' feature, ensuring they are visually clear and intuitive.
